Project Title:
QuickCart

Project Description:
QuickCart is a minimal and responsive E-commerce cart system where users can 
browse products, manage cart quantities, and simulate the checkout process. It 
focuses on essential cart functionality using React, HTML, CSS, and public APIs.

Live Demo:
https://ecommerce-three-delta-47.vercel.app/

Key Features:

🛒 Add to Cart: Users can add products from the home page to their cart 
 
●  🔁 Update Quantity: Increase or decrease item quantity directly in the cart 
 
●  💰 Real-time Total: Auto-updating total price display as items are 
added/updated 
 
●  📦 Checkout Simulation: Dummy checkout page to simulate order 
completion flow 
 
●  📱 Responsive Design: Works smoothly on devices of all screen sizes 

Tech Stack:

●  HTML5 – Structure of the pages 
 
●  CSS3 – Layout and styling 
 
●  JavaScript – Logic and interactivity 
 
●  React.js – Component-based UI structure 
 
●  React Router DOM – Page navigation 
 
●  Mock API – Product data and checkout 
simulation 
 
●  Git & GitHub – Version control 
 
●  Vercel – Deployment platform












Note: Most of the CSS was written manually, with occasional use of **GitHub Copilot** for class suggestions and layout ideas.
